Welcome to Wallet Tracker

Track your cryptocurrency wallets with ease!

Learn More

Features

Wallet Management

Effortlessly add and delete cryptocurrency wallets.

Real-time Tracking

Receive real-time updates on your wallet transactions.

Notifications

Get notified about incoming and outgoing transactions.

Prerequisites

  • Python 3.x installed
  • Flask and Telegram libraries
  • BlockCypher and Solana API keys

Setup

  1. Clone the repository
  2. Install required Python packages
  3. Set up BlockCypher and Solana API keys
  4. Run the Flask application

Usage

Follow these steps to start using Wallet Tracker:

  1. Add your cryptocurrency wallet
  2. Receive real-time updates
  3. Delete wallet if needed
